# Frontend Instructions for Surya AI

## Project Goal
Build the frontend interface for Surya AI. The layout and user experience must mimic the Gemini web interface.

## Tech Stack
* Framework: Next.js.
* Styling: Tailwind CSS.
* Icons: Lucide React.
* Fonts: Inter or SF Pro.

## Design System
* Create a highly professional design.
* Keep it clean. Match the aesthetic of Apple, Notion, or Linear.
* Use deep orange as the main primary color. Apply this to main buttons, active states, and focus rings.
* Support both light and dark modes.

## Layout Structure
* Build a collapsible sidebar on the left.
* Display chat history sessions in the sidebar.
* Place a "New Chat" button at the top of the sidebar.
* Create a main chat display area in the center.
* Pin the message input box to the bottom of the main area.
* Include send, microphone, and image upload icons in the input area.

## Core Features & UX
* Ensure the code runs fast and is highly optimized.
* Make the design fully responsive. It must work perfectly on mobile phones.
* Add clear loading states. Use a pulsing animation or skeleton loader when the AI is generating a response.
* Parse and render markdown in chat messages.
* Include syntax highlighting for code blocks.
* Implement auto-scroll to keep the latest message visible.
